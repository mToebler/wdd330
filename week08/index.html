Transform and Transitions

Transform
Transform property; handles these four functions: - translate: moves elements (x,y) - scale: controls element sizing (factor) - rotate: in deg or rad (360deg) - skew: in deg or rad along axis (x,y)
translate(x,y)
Moves elements left and right on the x axis, up and down on the y axis. transform: translate(1rem, 1rem); Be careful with translate; elements can move right off viewport. transform: translate(-20rem, 3rem);
scale(#)
All about scaling factor, takes no unit. transform: scale(2);
rotate(angle)
Rotates around center point rooted at original location, pre-transform. So if combined with translate will rotate along an arc. transform: rotate(45deg); transform: rotate(0.8rad) translate(15px, 15px);
skew(angle, angle)
Tilts the element along a 3D axis in degrees or radians (x,y) transform: skew(5deg, 5deg); Be careful with skew! Elements can be skewed into unviewable profiles. transform: skew(90deg);
Transition
transition property attaches qualifiers on transformed elements. - Transitions work with number based properties colors, positions, widths, %, heights, sizes - Text or call out values won't transition. font, callouts values: border styles, text decoration, etc. The list of transition properties: transition-property: (default: all) transition-duration: (how long from A=>B) transition-timing-function: ease-out transition-delay: (delay before transition) Shorthand transition: transition: transform 0.2s ease-in 50ms; The transforms above all have a transition duration of .5s

element.addEventListener('touchmove', (e)=>{ // grab the location of touch let loc8ion = e.targetTouches[0]; // assign element new coordinates based on the touch. element.style.left = loc8ion.pageX + 'px'; element.style.top = loc8ion.pageY + 'px'; });
second:
element.addEventListener('touchend', (e)=>{ // grab coordinates when released let x = parseInt(element.style.left); let y = parseInt(element.style.top); });
