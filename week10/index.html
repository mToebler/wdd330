Mobile Finger Jam w Canvas
Inspired by Wes Bos After diving into Canvas a bit more, I've decided it's not worthless at all. This is a Wes Bos exercise that I ran with and made mobile friendly, works on both, desktop and mobile, but reload if you switch between. I've disabled default mouseevents to prevent scrolling while finger jamming. Check out : finger jam (try slow, small circles) Canvas, A Quick How to:
// element with explicit width and height <canvas id='draw' width='800' height='800'>No C Msg</canvas> const canvas = document.querySelector('#draw'); // set context by getting it (ok values: '2d', '3d') // context: drawing magic happens const ctx = canvas.getContext('2d'); // set w/h to entire visible window canvas.width = window.innerWidth; canvas.height = window.innerHeight; // set base settings: ctx.strokeStyle = '#BADA55'; // for 1 color ctx.lineJoin = 'round'; // or bevel or miter fun to leave off ctx.lineCap = 'round'; // perhaps same values as join ? // drawing flag let isDrawing = false; // starting point of line to be drawn let lastX = 0; let lastY = 0; //function to draw function draw(event) { if(isDrawing) { console.log(event); lastX = lastX === 0 ? event.offsetX : lastX; lastY = lastY === 0 ? event.offsetY : lastY; // signals draw begin ctx.beginPath(); ctx.moveTo(lastX, lastY); ctx.lineTo(event.offsetX, event.offsetY); // ES 6 destructuring: [lastX, lastY] = [event.offsetX, event.offsetY]; // signlas draw end ctx.stroke(); } else { // destructuring [lastX, lastY] = [0, 0]; } } // event listeners to bring it all together. canvas.addEventListener('mousemove', draw); canvas.addEventListener('mousedown', () => isDrawing=true); canvas.addEventListener('mouseup', ()=> isDrawing = false); canvas.addEventListener('mouseout', ()=> isDrawing = false); Adapted from Wes Bos.
