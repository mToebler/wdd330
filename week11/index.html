Display Console Log on Mobile Pages
Adapted and improved from Moz MDN Web Docs Came across this little script to display logged messages in a mobile window. I've adapted it to allow multiple arguments, like conosle.log itself. < -- scroll -- >
// Whatever gets sent to this method also gets sent to the console.log. // a little resting going on here ... function logs(...msgs) { const logD = document.querySelector('#logs'); try { for(msg = 0; msg < msgs.length; msg++) logD.innerHTML += msgs[msg] + ' '; if(msgs.length > 0) logD.innerHTML += "\n"; // and here console.log(...msgs); } catch(err) { // will throw error if can't handle it. console.error('week11.js logs:', err); } }
Give it a try: logs('Testing testing', '1', '2', '3');
