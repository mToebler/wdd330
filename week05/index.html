Closures

Controlling Scope
Closers control what is and isn't in function scope. Last week, I used a closer in the WeakMap example of OOP encapsulation. With closures, inner function can gain access to the additional, supplied scope of the en-closing function. This allows for: - Encapsulation - Data privacy Classic example:
const getSecret = (secret) => { return { get: () => secret }; }; const key = getSecret('private'); key.get(); // returns private
The privleged method get() will grant access to secret set via with the returned object. Next, we'll explore partial application and currying with closures.
